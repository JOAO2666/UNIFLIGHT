## 🎮 **Como Jogar**

### **Controles Player 1:**
- **A/D**: Mover esquerda/direita
- **E**: Soco (+10 poder)
- **R**: Chute (+15 poder)
- **Q**: Poder Especial (requer 50 pontos)

### **Controles Player 2:**
- **←/→**: Mover esquerda/direita
- **O**: Soco (+10 poder)
- **I**: Chute (+15 poder)
- **P**: Poder Especial (requer 50 pontos)

## 🛠️ **Compilação**

### **Requisitos:**
- Raylib 5.0+
- Compilador C (GCC recomendado)
- Windows/Linux/macOS

### **Compilar:**
```bash
# Usando o script fornecido
./compile_notepad.bat

# Ou manualmente
gcc unifight.c -lraylib -lGL -lm -lpthread -ldl -lrt -lX11 -o unifight
```
